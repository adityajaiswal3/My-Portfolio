# 3D Animated Portfolio

A premium, state-of-the-art 3D portfolio website built with modern web technologies.

## ✨ Features

- **Immersive 3D Background**: Procedurally generated 3D shapes using `Three.js` and `@react-three/fiber`.
- **Glassmorphic UI**: Sleek, modern design with backdrop blurs and subtle borders.
- **Dynamic Animations**: Smooth transitions and scroll-triggered animations powered by `Framer Motion`.
- **Responsive Design**: Optimized for all screen sizes from mobile to desktop.
- **Performance Optimized**: Lazy-loaded 3D components and optimized asset delivery.

## 🚀 Tech Stack

- **Core**: React 19 + Vite
- **3D Engine**: Three.js
- **React 3D**: @react-three/fiber & @react-three/drei
- **Animations**: Framer Motion
- **Styling**: Vanilla CSS (Custom Variable System)
- **Icons**: Lucide React

## 🛠️ Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## 📂 Project Structure

- `src/components/Experience.jsx`: Contains the 3D scene and shapes.
- `src/App.jsx`: Main application layout and sections.
- `src/index.css`: Design system and global styles.
- `src/main.jsx`: Entry point.
